# TriHaRd: Higher Resilience for TEE Trusted Time

This repository contains the source code, as well as deployment and analysis scripts, for the "TriHaRd: Higher Resilience for TEE Trusted Time" paper, submission #135 at the ISSRE'25 Research Track.

## Contents

- [trihard](trihard) contains the main source code ([trihard/node](trihard/node)), analysis and plot generation scripts ([trihard/analysis](trihard/analysis));
- [deployment](deployment) contains VM deployment and setup scripts, with Azure SGX2 VMs as targets.
